# F-KAN

The simulation datasets used in this study are generated based on the following publicly available implementations:\\
Simulation Data A: adapted from https://github.com/caojiguo/FNN\\
Simulation Data B: adapted from https://github.com/shanghongxie/FSVM\\

All real-world datasets supporting the findings of this study are publicly accessible:\\
CGL dataset: https://eigenvector.com/resources/data-sets\\
Tecator dataset: available in the R package fda.usc\\
AA dataset: https://github.com/Mahsa000/Raman-Amino-Acids-Quantification\\
Milk dataset: https://github.com/SzymonUrbas/bplsr\\

The implementations of the main models are based on the following repositories:\\
FuncNN: https://github.com/caojiguo/FNN\\
FSVM: https://github.com/shanghongxie/FSVM\\
KAN: https://github.com/KindXiaoming/pykan\\

For data preprocessing and baseline methods:\\
Functional principal component analysis (FPCA), curve smoothing, and traditional machine learning models—including linear regression (e.g., OLS) and support vector machines (SVM)—are implemented using the scikit-learn library.\\
Functional Linear Models (FLM) and Functional Generalized Additive Models (FGAM) are primarily implemented using the R package refund.

The remaining code will be released in future updates.
